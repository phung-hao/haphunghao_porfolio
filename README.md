# haphunghao_porfolio
Hà Phụng Hảo - Professional Portfolio

A professional portfolio website designed to showcase competencies, academic journey, and real-world projects in E-commerce and Product Management.

🚀 Key Features

Visual Competency Reporting: Integrated Chart.js to display Radar charts (Professional Skills) and Bar charts (Academic Grades) transparently.

Smart Project Catalog: Project filtering system by categories: E-commerce, Product, and Marketing.

Brand Identity: Custom logos/avatars for each live project (Earlie, VNU Connect X, Caffinus, etc.).

Responsive Design: Fully optimized layout for all devices (Mobile, Tablet, Desktop) using Tailwind CSS.

Minimalist Aesthetic: Professional Pastel Pink & Neutral color palette, focusing on content and factual data.

🛠 Technologies Used

Language: HTML5.

Styling: Tailwind CSS (Utility-first CSS framework).

Charts: Chart.js (JavaScript charting library).

Typography: Inter Font (Google Fonts).
